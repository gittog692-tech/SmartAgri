# Smart Agri Intelligence & Supply Chain

A full-stack web application designed to help farmers make better agricultural decisions and connect directly with buyers, reducing middlemen in the supply chain.

## Prerequisites
- Node.js installed

## Installation & Setup
## 1. Clone the repository
```bash
git clone https://github.com/gittog692-tech/SmartAgri.git
cd SmartAgri
``` 
## 2. Setting up the Backend
```bash
cd backend
npm install
npm start
```
The backend server will run on `http://localhost:5000`.

## 3. Setting up the Frontend
Open another terminal and run the following commands:
```bash
cd frontend
npm install
npm run dev
```
The frontend vite server will typically start on `http://localhost:5173`. Open this URL in your web browser.

## Features
- **Crop Advisor:** Provides personalized crop recommendations based on soil type, weather, season, and location. Also offers smart farming tips.
- **Market Connect:** Eliminates middlemen by allowing farmers to list crops with expected prices directly. Buyers can contact farmers.
- **Government Schemes:** Displays current agricultural support schemes and benefits (like PM-KISAN, KCC, PMFBY).
- **Farmer Community:** A dedicated discussion page where farmers can ask queries and communicate with each other. Users can edit/delete their own data.
